# FriendlyBet

## Web based app for friendly betting. No prizes for winners. Just for fun!

### Version 1.0 feature collection target:
1) Authentication system
    - Login
    - Register
    - Verify email
    - Log out
    - Forgot password
    - Session expiration logic
2) Home page
3) Football matches from Premiere League
4) Formula 1
5) User's dashboard
